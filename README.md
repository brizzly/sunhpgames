# HP48G(X) Graphic Calculators

Very fast games I did in assembly language, 1995~1998

![HP48 Calculator](assets/hp48calc.jpg)

This was the best awesome and powerful graphic calculator at the time and one could build very fast games using assembly language ([See my ebook about SATURN ASM programming](assets/hp/hp48_assembler_julienmeyer_monlivre.docx)) even with 4 levels of grayscales graphics and fast scrollings. I was so impressed by the fast gameboy-like graphics game made possible on this great calculator, and also because I did scientific/eng studies that I bought one.

[I wrote so many games for this HP48 graphic calculators](https://www.hpcalc.org/authors/8): plenty of time and a memory card for backups! We still find those oldies calcs on eBay and also a great list of Emulators available out there: Emu48, x48 for Desktop and smartphone. I even try to build online emu48 with Javascript: [jsEmu48](http://www.jmapp.com/jsemu48).
